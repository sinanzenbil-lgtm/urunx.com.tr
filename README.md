# Urunx - Stok Takip Sistemi

Urunx, küçük ve orta ölçekli işletmeler için geliştirilmiş, hızlı ve modern bir stok takip uygulamasıdır.

## Özellikler

- **Hızlı Stok Takibi:** Barkod okuyucu desteği ile saniyeler içinde ürün ekleme ve çıkarma.
- **Toplu İşlemler:** Excel ile ürün yükleme ve toplu stok çıkışı.
- **Anlık Arama:** Binlerce ürün arasında milisaniyeler içinde arama yapma.
- **Detaylı Raporlama:** Ürün hareketlerini tarihçeli olarak izleme.
- **Kullanıcı Dostu Arayüz:** Modern, karanlık mod destekli ve mobil uyumlu tasarım.

## Kurulum

Bu proje [Next.js](https://nextjs.org/) ile geliştirilmiştir.

```bash
npm install
npm run dev
```

## Teknolojiler

- Next.js 14+ (App Router)
- Tailwind CSS
- Zustand (State Management)
- Lucide React (Icons)
- Radix UI
